# TileFocus

`TileFocus` は、macOS向けのスマートで柔軟なウィンドウマネージャーアプリです。特定のウィンドウに集中するための「フォーカスモード」や、不要なウィンドウを一時的に退避させておく「格納（Stage）機能」を備えています。

## 主な機能

### 1. フォーカスモード (Focus Mode)
* 選択した「マスターウィンドウ」を画面中央（または左右）に大きく配置し、その他のウィンドウをサイドバーに積み重ねるように配置します。
* 以下の多彩なレイアウトスタイルに対応しています：
  * **Centered**: 中央にメイン、左右にサブウィンドウを配置
  * **Left Main / Right Main**: 左右どちらかにメインを寄せ、反対側にサブウィンドウを配置
  * **Split Centered**: 中央にメインウィンドウを2つ並べ、両サイドにサブウィンドウを配置
  * **Absolute Split (2分割/3分割)**: 画面を完全に対等に2分割または3分割し、収まりきらないウィンドウは最小化（Dock格納）します。

### 2. ステージ・格納機能 (Stage & Unstage)
* 使用頻度の低いウィンドウを画面外（またはDock）へワンクリックで一時退避（格納）できます。
* 画面上部にマウスをホバーすると展開する「フローティングバー（上部バー）」から、格納中・表示中の全ウィンドウを一覧で視認・復帰できます。

### 3. 直感的な操作性
* **ドラッグ＆ドロップによる並べ替え**: フローティングバー内でウィンドウカードをドラッグ＆ドロップし、サイドウィンドウのレイアウト順序を簡単に並べ替えられます（マスターウィンドウを除く）。
* **ショートカットキー**:
  * `⌃⌘M` (Control + Command + M): 現在アクティブなフロントウィンドウを瞬時にマスターウィンドウに昇格します。
  * メニューバーアイコンからもモード変更やマスターウィンドウの切り替えが可能です。

## システム要件
* macOS 13 (Ventura) 以上 (推奨)
* アクセシビリティ（アクセシビリティAPIへのアクセス）権限の許可が必要です。

## セットアップ / ビルド方法
Swift Package Manager (SPM) プロジェクトとして構築されています。

1. リポジトリをクローンします。
2. ターミナルでリポジトリ直下に移動し、以下のコマンドでビルドします：
   ```bash
   swift build
   ```
3. もしくは、Xcodeで `Package.swift` を開き、ビルド・実行してください。

## ダウンロードしたアプリが起動しない / アクセシビリティ許可ができない場合の対処法

リリース（.zip）からダウンロードした `TileFocus.app` を実行する際、macOSのセキュリティ保護（Gatekeeper）やプライバシー制限により、正常に起動できなかったりアクセシビリティの許可が与えられなかったりする場合があります。

その場合は、以下の対処法をお試しください。

### 1. アプリケーションが「開発元を検証できないため開けない」場合
macOSのセキュリティチェックにより起動がブロックされている可能性があります。

* **対処法 A (推奨): 右クリックから開く**
  1. Finderで `TileFocus.app` を右クリック（または `Control` キーを押しながらクリック）し、「開く」を選択します。
  2. 警告ダイアログが表示されたら「開く」をクリックします。一度この操作を行えば、以降は通常のダブルクリックで起動できるようになります。
* **対処法 B: ターミナルで検疫属性 (Quarantine) を解除する**
  ターミナルを開き、以下のコマンドを実行してインターネットからダウンロードした属性フラグをクリアします：
  ```bash
  xattr -cr /path/to/TileFocus.app
  ```

### 2. Apple Silicon (M1/M2/M3) 環境でアプリが起動直後に強制終了する場合
ダウンロード時の破損や未署名状態によるクラッシュの可能性があります。ターミナルで以下のアドホック署名を再適用してください：

```bash
codesign --force --deep --sign - /path/to/TileFocus.app
```

### 3. アクセシビリティの許可リストに登録できない・チェックを入れても効かない場合
単一バイナリファイルのままだとアクセシビリティ許可が認識されません。必ずリリースされたZIPを解凍し、**`TileFocus.app` というアプリケーションパッケージの状態**で「システム設定 ➔ プライバシーとセキュリティ ➔ アクセシビリティ」のリストにドラッグ＆ドロップして許可を付与してください。
