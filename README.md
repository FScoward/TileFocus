# TileFocus

`TileFocus` は、macOS向けのスマートで柔軟なウィンドウマネージャーアプリです。特定のウィンドウに集中するための「フォーカスモード」や、不要なウィンドウを一時的に退避させておく「格納（Stage）機能」を備えています。

## 主な機能

### 1. フォーカスモード (Focus Mode)
* 選択した「マスターウィンドウ」を画面中央（または左右）に大きく配置し、その他のウィンドウをサイドバーに積み重ねるように配置します。
* 以下の多彩なレイアウトスタイルに対応しています：
  * **Centered**: 中央にメイン、左右にサブウィンドウを配置
  * **Left Main / Right Main**: 左右どちらかにメインを寄せ、反対側にサブウィンドウを配置
  * **Split Centered**: 中央にメインウィンドウを2つ並べ、両サイドにサブウィンドウを配置
  * **Absolute Split (2分割/3分割)**: 画面を完全に対等に2分割または3分割し、収まりきらないウィンドウは最小化（Dock格納）します。

### 2. ステージ・格納機能 (Stage & Unstage)
* 使用頻度の低いウィンドウを画面外（またはDock）へワンクリックで一時退避（格納）できます。
* 画面上部にマウスをホバーすると展開する「フローティングバー（上部バー）」から、格納中・表示中の全ウィンドウを一覧で視認・復帰できます。

### 3. 直感的な操作性
* **ドラッグ＆ドロップによる並べ替え**: フローティングバー内でウィンドウカードをドラッグ＆ドロップし、サイドウィンドウのレイアウト順序を簡単に並べ替えられます（マスターウィンドウを除く）。
* **ショートカットキー**:
  * `⌃⌘M` (Control + Command + M): 現在アクティブなフロントウィンドウを瞬時にマスターウィンドウに昇格します。
  * メニューバーアイコンからもモード変更やマスターウィンドウの切り替えが可能です。

## システム要件
* macOS 13 (Ventura) 以上 (推奨)
* アクセシビリティ（アクセシビリティAPIへのアクセス）権限の許可が必要です。

## セットアップ / ビルド方法
Swift Package Manager (SPM) プロジェクトとして構築されています。

1. リポジトリをクローンします。
2. ターミナルでリポジトリ直下に移動し、以下のコマンドでビルドします：
   ```bash
   swift build
   ```
3. もしくは、Xcodeで `Package.swift` を開き、ビルド・実行してください。
